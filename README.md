# Auth-Page

A lightweight registration interface built with HTML5, CSS3, and Vanilla JavaScript featuring real-time client-side validation.

### ✨ Key Features

* **Validation**: Uses Regex for secure passwords, CNIC (XXXXX-XXXXXXX-X), and Pakistani phone formats (03XXXXXXXXX).
* **UI/UX**: Responsive Flexbox layout with a modern blue-themed interface.
* **Error Handling**: Dynamic messages for username, email, and gender selection.
* **Clean Code**: No external libraries or frameworks required.

### 🛠️ Validation Logic

* **Username**: Minimum 4 characters.
* **Password**: Requires 1 uppercase, 1 number, and 1 special character.
* **Formatting**: Strict patterns for CNIC and mobile numbers.

### 🚀 Setup

1. Clone the repository.
2. Open `registration-authentication-Page.html` in any modern web browser.
